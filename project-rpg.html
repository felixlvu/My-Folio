RPG

The aim of this project was to create an RPG in C using the 'CSFML' graphics library. We were to create it in groups of 3, including a story and a main character controllable by the user. The key elements we had to include in this RPG were as follows:

-A menu offering settings with buttons for playing and quitting.
-A multi-dimensional map allowing you to pass behind objects.
-A storyline.
-Enemies and an ending.
-Dialogue with non-playable characters.
