API

The aim of the personal project was to create an API enabling communication between the terminal and a C-language program in order to test my programs. This allows someone who hasn't downloaded a Linux environment to try out my programs.
This API is currently used to test the BSQ and 42sh projects. This was the first time I'd been interested in APIs, and it was really interesting to understand how they work and their advantages in a site.
Thanks to this personal project, I was able to discover many new possibilities, although I had already experimented with routes and the JavaScript language with the epytodo project.
